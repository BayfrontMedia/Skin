Typography

The tu-typo utility class is used to give default typography styling to some base elements.

Any child elements inside tu-typo will inherit these styles. To prevent a specific element from inheriting the typography styling, assign it the tu-typo-omit class. To apply the typography styling to a specific element not inside a tu-typo element, assign it the tu-typo-apply class.
