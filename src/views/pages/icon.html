Icons are available as a web component using the skin-icon tag. Icons exist for each Heroicon outline style.

Attributes:

  • name: Valid Heroicon name (required)
  • data-class: Class(es) to add to svg (optional)
  • data-stroke-width: Stroke width, 1.5 by default (optional)
