Quotes are available as a web component using the skin-quote tag. They can contain any content, but would typically contain a blockquote, and a cite.

Attributes:

  • data-float: Float (optional)
  • data-style: Alternate style (optional)

Valid floats include:

  • left: Float left starting with the medium breakpoint
  • center: Float center (center align text) starting with the medium breakpoint
  • right: Float right starting with the medium breakpoint

Valid styles include:

  • quote: Large double quote
