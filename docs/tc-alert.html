Web component

Alerts are available to be used as a web component. Any tc-alert-* utility class can be applied.

Attributes:

  • id: Unique ID (required if dismissible)
  • data-icon: Any valid skin-icon name (optional)
  • data-dismiss-duration: Minutes to keep alert hidden after dismissed (optional)
